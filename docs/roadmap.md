# RuSSH Roadmap

- `0.1`: workspace bootstrap + API skeleton + parser/codec basics.
- `0.2`: transport negotiation and rekey state machine implementation.
- `0.3`: auth engine (pubkey/password/keyboard-interactive) with cert validation primitives.
- `0.4`: channels, forwarding, and file-transfer protocol kernels.
- `0.5`: advanced features (agent forwarding, jump chain, connection multiplexing).
- `0.6`: hardening (fuzzing/interop/perf), API stabilization, release candidates.
- `1.0`: security review gate and stable release.
